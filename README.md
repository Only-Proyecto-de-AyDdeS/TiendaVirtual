#  Tienda Virtual de ONLY  

##  Descripción General  

Este repositorio contiene el desarrollo básico de una **tienda virtual** perteneciente a la **cadena de almacenes ONLY**.  
El proyecto tiene como objetivo aplicar los principios fundamentales del desarrollo de software orientado a objetos, la programación modular y las buenas prácticas de ingeniería de software en un entorno colaborativo.  

A través de este sistema, se busca simular el funcionamiento de una tienda en línea, donde los usuarios pueden **visualizar productos**, **realizar compras** y **gestionar el inventario**, según las funcionalidades implementadas en esta versión.
